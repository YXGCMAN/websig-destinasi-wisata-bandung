# 🌍 Web SIG Destinasi Wisata Kota Bandung

## Deskripsi

Web SIG Destinasi Wisata Kota Bandung merupakan aplikasi berbasis web yang digunakan untuk menampilkan persebaran lokasi wisata di Kota Bandung secara interaktif menggunakan peta digital.

Aplikasi ini dibangun menggunakan Python Flask sebagai backend, Leaflet sebagai library peta, OpenStreetMap sebagai basemap, serta dataset destinasi wisata Kota Bandung dalam format CSV.

---

## Fitur

- Menampilkan peta interaktif
- Marker lokasi wisata
- Popup informasi wisata
- Pencarian destinasi wisata
- Filter berdasarkan kategori
- Statistik jumlah wisata
- Daftar seluruh destinasi wisata
- Responsive Web Design
